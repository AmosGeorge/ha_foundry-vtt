# FoundryVTT for HomeAssistant
FoundryVTT local addon for HomeAssistant

## Install

Download the linux version of Foundry VTT. Rename the zip file `foundryvtt.zip`.

In HomeAssistant:
- create a folder `~/addons/foundry`
- create a folder `~/addon_configs/local_foundry/`
- drop this repository's files and `foundryvtt.zip` in `~/addons/foundry`

Go to **Home Assistant → Settings → Add-ons → Add-on Store**
**Menu ⋮ → Reload / Check for updates**

The FoundryVTT module should appear in *Local apps* section.
